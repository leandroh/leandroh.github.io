Instalando Elixir usando asdf

Plugin Elixir para gerenciador de versões asdf.
junho 26, 2022

Anteriormente, falamos sobre o asdf, e vamos instalar a linguagem Elixir usando esse gerenciador de versões de linguagens de programação chamado asdf.

Instalando#

Vamos precisar do plugin Elixir asdf-elixir para o asdf, este plugin depende do unzip instalado na sua máquina. Mas antes, precisamos instalar o Erlang. E para instalar as versões do Erlang, usamos o plugin asdf-erlang.

Com o asdf instalado na sua máquina, no seu terminal, rode o comando:

asdf plugin-add erlang

E, logo em seguida, rode o comando:

asdf plugin-add elixir

Com o plugin adicionado, precisamos instalar uma versão. Com o comando:

asdf install erlang latest

E também:

asdf install elixir latest

No lugar da palavra latest (que pega a última versão), você pode passar a versão desejada. Para listar as versões disponibilizadas, rode o comando asdf list elixir:

Instale uma versão desejada, desta forma:

asdf install elixir 1.13.3

Definindo uma versão#

Agora você vai precisar definir uma versão local ou global para a sua nova linguagem. Para definir uma versão local do diretório atual, rode o comando:

asdf local elixir 1.13.3

Para uma versão instalada globalmente em todo o sitema, rode:

asdf global elixir 1.13.3
